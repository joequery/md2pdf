md2pdf
======

Markdown to PDF. This script uses an OS system call because I was too
lazy to figure out the xhtml2pdf API.

Requirements
------------

* Python 2.6+

Installation
------------

    $ git clone git://github.com/joequery/md2pdf.git
    $ cd md2pdf
    $ sudo python setup.py install

Usage
-----

    $ md2pdf some-document.md
